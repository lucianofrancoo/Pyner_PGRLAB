                        📊 RELEVANCE SCORE CALCULATION
═══════════════════════════════════════════════════════════════════════════════

The LLM (Qwen 3.5) evaluates the RELEVANCE of the paper by comparing it with the USER QUERY.

═══════════════════════════════════════════════════════════════════════════════
EXAMPLE 1: Paper 41068586
═══════════════════════════════════════════════════════════════════════════════

USER QUERY:
┌─────────────────────────────────────────────────────────────────────────────┐
│ ("Solanum lycopersicum"[Organism] OR "tomato") AND ("RNA-Seq")              │
└─────────────────────────────────────────────────────────────────────────────┘

PAPER INFO:
┌─────────────────────────────────────────────────────────────────────────────┐
│ Title: "Transcriptome-based meta-analysis of drought stress regulatory      │
│        genes in tomato."                                                     │
│                                                                              │
│ Abstract: "Plants possess molecular defense systems... The RNA-seq           │
│          technology assists biologists in quickly identifying genes..."      │
│                                                                              │
│ Full Text Methods:                                                           │
│ "We searched public databases for drought-responsive genes in               │
│  Solanum lycopersicum, Solanum tuberosum, and Nicotiana species.             │
│  Expression profiles were analyzed using RNA-seq data from..."              │
└─────────────────────────────────────────────────────────────────────────────┘

LLM ANALYSIS:
┌─────────────────────────────────────────────────────────────────────────────┐
│ CRITERION 1: Does ORGANISM (Solanum lycopersicum / tomato) appear?          │
│ ✓ MATCH in: Title, Abstract, Full Text                                      │
│   - Title: "...tomato"                                                       │
│   - Abstract: "...tomato..."                                                 │
│   - Methods: "Solanum lycopersicum", "Solanum tuberosum"                    │
│ → Score: 10/10                                                               │
│                                                                              │
│ CRITERION 2: Does TECHNIQUE (RNA-Seq) appear?                               │
│ ✓ MATCH in: Title, Abstract, Full Text                                      │
│   - Title: "Transcriptome-based meta-analysis"                              │
│   - Abstract: "RNA-seq technology"                                           │
│   - Methods: "Expression profiles were analyzed using RNA-seq"              │
│ → Score: 10/10                                                               │
│                                                                              │
│ CRITERION 3: Is the paper FUNDAMENTALLY about this?                         │
│ ✓ YES - It is a meta-analysis specifically about drought genes in tomato    │
│       using RNA-seq.                                                         │
│ → Score: 10/10                                                               │
│                                                                              │
│ FINAL SCORE: (10 + 10 + 10) / 3 = 10/10  ← PERFECTLY RELEVANT               │
└─────────────────────────────────────────────────────────────────────────────┘

OUTPUT CSV:

  Is_Relevant: Yes (since 10 ≥ 5)

═══════════════════════════════════════════════════════════════════════════════
VARIABLES AFFECTING THE SCORE
═══════════════════════════════════════════════════════════════════════════════

1. QUERY MATCH
   ✓ Paper mentions "tomato" → +1.5
   ✓ Paper mentions "drought" → +1.5
   ✓ Paper mentions "RNA-Seq" → +2.0
   ✓ Match in Title vs. Body → different weights
   
   Logic: If query seeks (A AND B), the paper must have BOTH.

2. SPECIFICITY
   ✓ If paper is ABOUT the topic → +3.0
   ✓ If paper MENTIONS the topic → +1.5
   ✓ If paper is tangential → +0.5
   
   Logic: A paper titled "Tomato Transcriptome Analysis" is better than
          "Plant Stress Responses" even if it mentions tomato.

3. COMPLETENESS
   ✓ Paper covers all criteria → 10/10
   ✓ Paper covers 80% criteria → 8/10
   ✓ Paper covers 50% criteria → 5-6/10
   ✓ Paper covers <50% → <5/10
   
   Logic: The more criteria match, the higher the score.

4. METHODS USED
   ✓ Paper uses exact technique from query → +2.0
   ✓ Paper uses similar technique → +1.5
   ✓ Paper uses related technique → +1.0
   ✓ Paper DOES NOT mention technique → 0 in this category
   
   Logic: If the query specifies RNA-Seq but the paper only has microarray,
          the score decreases (but remains relevant).

═══════════════════════════════════════════════════════════════════════════════
Is_Relevant DECISION (Yes/No)
═══════════════════════════════════════════════════════════════════════════════

if Relevance_Score >= 5:
    Is_Relevant = "Yes"
else:
    Is_Relevant = "No"

Threshold configurable in: Data_Analyzer/config.py → RELEVANCE_THRESHOLD = 5

Papers with Score 5-6 are "borderline" relevant.
Papers with Score 0-4 are not considered relevant.
Papers with Score 7-10 are clearly relevant.
